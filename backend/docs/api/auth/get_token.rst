获取token
====

简要描述
-----------

通过用户名密码获取token。目前的token过期时间为一小时。

请求URL
-----------

- /apiv1/token
  
请求方式
------------------

- POST

参数
-------------

========  ====  ======  ====
 参数名   必选   类型   说明
========  ====  ======  ====
username  是    string
password  是    string
========  ====  ======  ====

返回示例
-----------

::

    {
        "token": "afdasfwagagadsfadfdafaaaaaa"
    }
